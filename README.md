# Devtools

Набор небольших браузерных инструментов для разработчиков.

## JSON и XML форматтер

- форматирование и минификация JSON и XML;
- форматирование и проверка синтаксиса Standard SQL;
- кодирование и декодирование JWT, включая HMAC-подпись HS256/HS384/HS512;
- анализ и генерация стандартных CRON-выражений с расчётом ближайших запусков;
- проверка синтаксиса с описанием ошибки;
- загрузка, перетаскивание, копирование и скачивание файлов;
- локальная обработка без отправки данных на сервер.
- интерфейс на украинском и английском языках.
- построчное сравнение текстов с настройками регистра и пробелов;
- смена регистра текста: верхний, нижний, заголовочный, предложения и инверсия;
- поиск кириллических символов в латинском тексте и замена визуально похожих символов.

Приложение не требует сборки. Для локального запуска:

```bash
python -m http.server 8000
```

Откройте `http://127.0.0.1:8000`.

## Azure Static Web Apps

При создании ресурса укажите:

- **App location:** `/`
- **Api location:** оставить пустым
- **Output location:** оставить пустым

Конфигурация маршрутов и заголовков находится в `staticwebapp.config.json`.

## Third-party

SQL форматируется библиотекой `sql-formatter` 15.8.2 (MIT). Копия лицензии находится в `vendor/sql-formatter/LICENSE`. CRON-выражения рассчитываются библиотекой `Croner` 10.0.1 и описываются библиотекой `cRonstrue` 3.24.0 (MIT); лицензии находятся в соответствующих каталогах `vendor/`.
